## Quick Start

Trading competitions web app built with Next.js App Router. Communicates with the API backend.

## Key Entry Points

- `app/page.tsx` - Home page
- `app/layout.tsx` - Root layout
- `components/` - Shared React components
- `app/api/sandbox/` - API proxy routes for sandbox competitions
